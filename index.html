What is SAND ?

a.k.a. ISO/IEC 23009 DASH part 5 Server And Network-assisted DASH

SAND defines a list of messages a DASH client and a SAND-enabled HTTP server (DANE) can exchange.

SAND defines protocols for a DANE and a DASH client to exchange the SAND messages

DANEs can improve the delivery of DASH segments and the DASH clients the user's experience.

Real-time metrics reporting via WebSocket

Every DASH player connects to the same DANE and sends metrics to it. The DASH metrics collected by each DASH client are sent over a WebSocket connection to the DANE. The goal is to allow a content provider, a CDN provider, to detect issues in the delivery of DASH content and react as quickly as posssible to remediate.
