What is SAND ?

A.k.a. ISO/IEC 23009 DASH part 5 SAND

SAND defines a list of messages a DASH client and a HTTP server can exchange.

SAND defines protocols for a HTTP server and a DASH client to exchange the SAND messages

By exhanging information, HTTP servers and DASH clients can improve their performances
